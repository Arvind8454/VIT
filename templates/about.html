About This Project

This application provides end-to-end image recognition using Vision Transformers, enhanced through self-supervised pretraining and explainable AI visualizations.

What The Project Does

The system classifies uploaded images, returns confidence scores, and explains predictions through heatmaps. It is designed for practical experimentation and interpretable model behavior.

Technology Used

  • Python, Flask, Streamlit
  • PyTorch, Hugging Face Transformers, timm
  • NumPy, Pandas, OpenCV, Matplotlib
  • CIFAR-10 training and evaluation pipeline

Vision Transformer Recognition

ViT splits images into patches, embeds them, and models global relationships through self-attention, enabling strong image classification performance.

Explainable AI

The app includes Grad-CAM and attention rollout to highlight which regions contribute to model decisions, helping users trust and debug predictions.
